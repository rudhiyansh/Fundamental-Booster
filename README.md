# Personal Data Collector

A simple interactive Python program that collects personal information from the user, displays the data along with its data type and memory address, and calculates the user's birth year based on their age.

## 📖 Project Description

This project is designed for beginners learning Python fundamentals. It demonstrates:

- User input handling
- Data type conversion (String, Integer, Float)
- Formatted output using f-strings
- Working with Python's built-in `type()` and `id()` functions
- Using the `datetime` module
- Basic arithmetic operations

The program asks the user for their name, age, height, and favorite number, then displays the collected information and estimates their birth year.

---

## ✨ Features

- Interactive command-line interface
- Collects personal information from users
- Demonstrates Python data types
- Displays memory addresses of variables
- Calculates birth year using the current year
